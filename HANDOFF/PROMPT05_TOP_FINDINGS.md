# Prompt #05 Top Findings

1. The previous design’s biggest weakness was not beauty; it was weak movement legibility.
2. Desktop maturity required a layout rebuild more than a style pass.
3. Stories become much more defensible when they stay tightly job-relevant and action-heavy.
4. Trust earns more power when it behaves like a changing system rather than a static explanation.
5. Saved searches and alerts are stronger when they predict the next useful change, not just summarize the current state.
