# Feed Layout Options Considered

## Option A: stacked modules with stronger styling

Rejected because it would preserve the structural weakness of the prior version.

## Option B: blended pulse feed mixing jobs, stories, and alerts into one list

Strong for novelty, but weaker for clarity tonight because the product still needs explicit trust and story distinction.

## Option C: delta stage + story reel + ranked jobs + rails

Chosen because it creates a clear operating system: movement first, media second, ranked action third.
